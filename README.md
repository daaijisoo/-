 二战考研复习驾驶舱

一个为我朋友制作的二战考研设计的 PWA 渐进式 Web 应用，帮助高效管理复习进度、掌握知识点、进行刷题练习。

## 🌟 核心功能

### 1. 复习驾驶舱（Dashboard）
- **考试倒计时** - 自定义考研日期，实时显示剩余天数
- **每日打卡清单** - 包含政治、英语、数学、专业课四大科目的学习任务
- **自定义任务** - 灵活添加个人学习计划
- **进度追踪** - 可视化显示每日任务完成率
- **番茄钟计时器** - 25 分钟专注学习 + 5 分钟休息，支持自定义时长
- **速记本** - 分科目快速记录学习要点和易错点
- **励志语录** - 随机显示激励语句，保持学习动力

### 2. 知识点库（Knowledge Base）
- **分科目管理** - 政治、英语、数学、专业课四大分类
- **增删查改** - 灵活添加、编辑、删除知识点
- **标签筛选** - 按科目快速筛选知识点
- **本地存储** - 数据保存在浏览器本地，无需联网

### 3. 题库系统（Quiz）
- **多种题型** - 支持选择题、填空题、大题
- **随机抽题** - 每次随机生成 15 道题进行练习
- **即时反馈** - 选择题和填空题提交后立即显示对错
- **答案解析** - 每道题都配有详细解析
- **正确率统计** - 实时显示本科目答题正确率
- **多科目切换** - 支持政治、英语、数学、专业课独立刷题

## 🚀 技术栈

- **前端框架**: 纯 HTML5 + CSS3 + JavaScript（零依赖）
- **数据存储**: localStorage（浏览器本地存储）
- **PWA 支持**: Service Worker + Web App Manifest
- **响应式设计**: 适配手机、平板、桌面端
- **离线可用**: 支持完全离线使用

## 📱 PWA 安装说明

### 手机端安装（推荐）
1. 使用 Chrome/Safari 打开应用
2. 点击浏览器菜单中的"添加到主屏幕"
3. 应用图标将出现在手机桌面，支持离线使用

### 桌面端安装
1. 使用 Chrome/Edge 打开应用
2. 点击地址栏右侧的安装图标
3. 按照提示完成安装

## 💡 使用技巧

### 番茄钟最佳实践
- 设置 25 分钟专注时间，5 分钟短休息
- 每 4 个番茄钟后，进行 15-30 分钟长休息
- 学习时关闭手机通知，保持专注

### 知识点管理建议
- 按科目分类整理知识点
- 定期复习已添加的知识点
- 使用简短、易记的描述

### 刷题策略
- 每天完成至少一组（15 道）练习题
- 重点关注错题和解析
- 定期切换科目，均衡复习
